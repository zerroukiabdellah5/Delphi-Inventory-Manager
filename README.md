# Product Management System (Delphi VCL)

## Overview
A Delphi-based desktop application logic designed for managing supermarket inventory. This project demonstrates backend integration with SQL databases using ADO components.

## Technical Skills
- **Delphi Pascal:** Clean and structured unit development.
- **Database Connectivity:** Using TADOQuery to execute SQL commands.
- **Parameterized Queries:** Secured database operations to prevent SQL injection.
- **Business Logic:** Implementation of CRUD (Create, Read, Update, Delete) operations.

## Why Delphi?
Delphi is a powerful tool for building fast, native Windows applications, which is essential for stable Point-of-Sale (POS) systems in retail environments.
